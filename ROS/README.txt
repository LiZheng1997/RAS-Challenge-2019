- You can determine if ROS is currently installed on the machine by typing "rosversion -d" in the terminal.

ROSinstall.bash
- If you have installed Ubuntu 16.04 LTS from the image provided from their website, you will need to run "bash ROSinstall.bash".
- If you have installed a ROS machine as a VM using the image we provided, you will not need to run "bash ROSinstall.bash". 
- You may need to modify the permissions of the file using "chmod +x ROSinstall.bash"
